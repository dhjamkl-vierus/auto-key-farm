❌ 1. Hotkey trong browser ≠ Global

👉 Hiện tại:

F1/F2/F3 chỉ hoạt động khi tab đang focus
Không đúng kiểu “tool hack thật”

✅ Fix chuẩn:

// Electron main process
globalShortcut.register("F1", () => toggleSystem());
globalShortcut.register("F2", () => toggleMenu());
globalShortcut.register("F3", () => app.quit());

👉 Nếu không lên Electron → chấp nhận limitation

❌ 2. AutoClick + TargetKey dễ bị “loop chồng”

👉 Nếu user spam ON/OFF nhanh → tạo nhiều interval

✅ Fix chuẩn:

let loop: NodeJS.Timeout | null = null;

export function startLoop(fn: () => void, delay: number) {
  if (loop) return; // 🔥 chặn duplicate

  loop = setInterval(fn, delay);
}

export function stopLoop() {
  if (!loop) return;
  clearInterval(loop);
  loop = null;
}
❌ 3. Debug ticker vẫn có nguy cơ spam nhẹ

👉 Bạn đã gộp msg, nhưng thiếu cooldown

✅ nâng cấp:

let lastTime = 0;

export function log(msg: string) {
  const now = Date.now();

  if (now - lastTime < 200) return; // 🔥 throttle

  lastTime = now;
  console.log(msg);
}

🧩 ⚙️ 3. MODE LOGIC (CHUẨN HOÁ)

👉 Hiện bạn đang:

check mode → show/hide UI

👉 Nhưng thiếu logic layer

✅ chuẩn:

export function applyMode(mode: string) {
  STATE.mode = mode;

  if (mode === "Easy") {
    STATE.weapon.mode = null;
  }

  if (mode === "Boss Rush" || mode === "Infinity Tower") {
    STATE.weapon.mode = "manual"; // default
  }
}
🎯 4. TARGET KEY SYSTEM (PRO VERSION)

👉 Bạn làm basic rồi, nhưng thiếu:

❌ lỗi phổ biến:
key không normalize
delay không apply riêng từng slot
✅ version xịn:
export function normalizeKey(key: string) {
  return key.length === 1 ? key.toUpperCase() : key;
}

export function handleTarget(slot) {
  if (!STATE.systemOn) return;

  setTimeout(() => {
    debug(`Trigger ${slot.key}`);
  }, slot.delay);
}
⚔️ 5. WEAPON SWITCH (LOGIC ĐÚNG)

👉 Hiện bạn mới UI, cần logic:

export function weaponLoop() {
  if (STATE.weapon.mode === "manual") {
    const [s1, s2] = STATE.weapon.slots;

    setTimeout(() => debug(`Slot ${s1}`), 0);
    setTimeout(() => debug(`Slot ${s2}`), STATE.weapon.delay);
  }

  if (STATE.weapon.mode === "auto") {
    debug("Auto switch");
  }
}

💾 8. STORAGE CHUẨN (ANTI CORRUPT)
export function loadSafe(key, fallback) {
  try {
    return JSON.parse(localStorage.getItem(key)) ?? fallback;
  } catch {
    return fallback;
  }
}
